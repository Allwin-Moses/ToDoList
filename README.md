# 📝 ToDo List App

A simple web-based ToDo list built using **ASP.NET MVC**, **C#**, **ADO.NET**, and **SQL Server**.

---

## 🚀 Features

- ➕ Add tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks  
- 🖊️ Edit tasks (if applicable)  
- 🧠 Stores data using SQL stored procedures

---

## 💻 Tech Stack

- ASP.NET MVC  
- C#  
- ADO.NET  
- SQL Server
